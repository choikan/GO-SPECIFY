# GO-SPECIFY

** 광운대학교 제 1회 파이썬 SW 활용 경진 대회 **
**** 패트와 매트 팀 ****

공공기관 IT 채용 공고 분석 기반 맞춤형 취업 준비 로드맵 서비스

---

## 📌 프로젝트 개요
**GO-SPECIFY**는 공공기관 IT 직무 채용 공고를 분석하여  
취업 준비생이 **어떤 역량을, 언제, 어떤 순서로 준비해야 하는지**를  
로드맵 형태로 제시하는 웹 기반 서비스입니다.

비정형 텍스트로 제공되는 채용 공고와 첨부 PDF 문서를 분석하여  
직무 요구 역량, 기술 스택, 자격증 및 시험 일정을 구조화하고,  
공고 마감일을 기준으로 **실질적으로 도움이 되는 준비 계획**을 제공합니다.

---

## 🎯 주요 기능

### 1️⃣ 공공기관 채용 공고 수집
- 공공데이터포털 API를 활용하여 채용 공고 목록 수집
- 공고 제목, 기관명, 근무지, 채용 구분, 학력, 고용 형태 등 필터 제공
- 캐시 기반 구조로 반복 실행 시 빠른 로딩 지원

---

### 2️⃣ 공고 상세 분석 (RAG 기반)
- 채용 공고에 첨부된 PDF 문서 자동 분석
- 비정형 텍스트에서 다음 정보 추출:
  - 직무명 및 주요 업무
  - 요구 기술 스택
  - 자격증 및 어학 요건
  - 가점 및 경력 조건
- RAG(Retrieval-Augmented Generation) 구조를 적용하여
  문서 기반 일관성 향상

---

### 3️⃣ 맞춤형 취업 준비 로드맵 생성
- 공고 분석 결과를 기반으로 **전공 연계 취업 준비 로드맵 자동 생성**
- 자격증 및 시험 일정 데이터와 연계
- 공고 마감일 이전 응시 가능한 시험만 선별하여 제공
- 로드맵 결과 PDF 다운로드 지원

---

### 4️⃣ 사용자 편의 기능
- 관심 공고(즐겨찾기) 기능
- 최근 생성 / 저장된 로드맵 관리
- 일부 공고 정보 누락 시에도 정상 동작하도록 예외 처리 설계
